# white_label_saas

Plataforma SaaS **white label** multi-empresa (especificação em [`docs/PROMPT-WHITE-LABEL-SAAS.md`](docs/PROMPT-WHITE-LABEL-SAAS.md)).

Stack inicial: **React + TypeScript + Vite**.

**Repositório:** [Carro-Assinatura/white_label_saas](https://github.com/Carro-Assinatura/white_label_saas)

## Desenvolvimento local

```bash
cd white_label_saas
npm install
npm run dev
```

## Abrir no Cursor

**File → Open Folder** → selecione a pasta `white_label_saas`.
